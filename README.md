# Iris Dataset - Data Visualization

## Objective
The objective of this project was to visualize patterns within the Iris dataset and compare different flower features using graphical representations.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook / VS Code

## Tasks Performed

### 1. Data Loading
- Imported the Iris dataset using Pandas.
- Examined the dataset structure using:
  - `head()`
  - `info()`

### 2. Bar Chart Visualization
- Created a bar chart to display the count of flowers in each species category.
- Compared the distribution of:
  - Iris-setosa
  - Iris-versicolor
  - Iris-virginica

### 3. Histogram Visualization
- Created a histogram for petal length.
- Analyzed the frequency distribution of petal measurements.

### 4. Scatter Plot Visualization
- Created scatter plots to compare:
  - Petal Length vs Petal Width
  - Sepal Length vs Petal Length
- Used species-based color coding for better interpretation.

### 5. Feature Comparison
- Compared relationships among multiple features.
- Utilized Seaborn's Pairplot for comprehensive feature analysis.

## Observations
- Petal length and petal width showed a strong positive relationship.
- Setosa species was clearly distinguishable from the other species.
- Versicolor and Virginica displayed partial overlap.

## Outcome
Meaningful visualizations were created to identify feature relationships and species-level patterns within the Iris dataset.
